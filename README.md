# Portfolio de Thomas

Ce site internet correspond au 1er exercice de la formation. 
Il a pour but de promouvoir le travail de Thomas, graphiste via les éléments suivants : présentation, blog, projets, formulaire de contact.

## Installation

Le package comprend :
* un fichier index.hml 
* un répertoire css avec la feuille de style
* un répertoire images avec les images du site web 

La police utilisée est Heebo de GoogleFonts. La police est directement installée dans le fichier html.

## Usage

Site vitrine statique utilisant les langages HTML5 et CSS.
Il a été validé via W3C et est développé pour être accessible aux personnes en situation de handicap.
